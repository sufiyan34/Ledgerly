
# Ledgerly 💰

A clean, intuitive expense tracker app built with Flutter to help you manage your personal finances, track spending, set budgets, and gain insights into your money habits.

## 📱 About

Ledgerly is a cross-platform mobile application that allows users to record income and expenses, categorize transactions, set monthly budgets, and visualize spending patterns through interactive charts and reports — all in one simple, beautifully designed interface.

## ✨ Features

- **Transaction Management** — Add, edit, and delete income/expense entries with categories, notes, and receipt attachments
- **Dashboard Overview** — View total balance, monthly summary, and recent transactions at a glance
- **Smart Categorization** — Organize transactions with customizable categories and icons
- **Budget Tracking** — Set monthly budget limits per category with visual progress indicators and alerts
- **Analytics & Reports** — Interactive pie charts, bar graphs, and trend analysis (weekly/monthly/yearly)
- **Search & Filters** — Quickly find transactions by date, category, or type
- **Authentication** — Secure login/signup with password recovery
- **Dark Mode** — Light and dark theme support
- **Local Data Storage** — Offline-first with local database (SQLite/Hive)
- **Multi-Currency Support** — Choose your preferred currency

## 🛠️ Tech Stack

- **Framework**: Flutter (Dart)
- **State Management**: Provider 
- **Local Database**: shared_preferences
- **Charts**: fl_chart
- **Authentication**: Firebase Auth (or local auth)
- **Date Handling**: intl
